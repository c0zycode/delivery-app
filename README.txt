# Project Title

Delivery App

## Getting Started

This is a simple program I created built using Python and sqlite3. It is used to track data about deliveries that are made. Ran through command line or terminal.

### Prerequisites

What things you need to install the software and how to install them

Python 3.8.3
https://www.python.org/downloads/

### Installing

First, open Delivery App folder.

Then, move 'deliveries.csv' from sample input into directory with app.py.

Open the terminal.

Navigate to the 'Delivery App' folder

Run the application:

python app.py
